Weather Forecast

Shows the current 7-Day Weather for any city.

The main purpose is to output the weather conditions of a particular city when you input a city name.

This application showcases fundamental React.js concepts: State Management, Class and Functional, Reusable Components, and JSX. Additionally, it connects to the Open Weather API using the Fetch API and the Open Weather API Key.

Burger Builder

Fast Food web application that allows you to Add and Remove toppings and Submit order.

A React.js web application that simulates a online order menu which allows customers to add, change, and remove # of toppings to a burger in real-time. As ingredients are added/remove, the sales price reflects those changes.

This shows knowledge of React core concepts: State Management, Reusable Components, React Hooks, and JSX.
